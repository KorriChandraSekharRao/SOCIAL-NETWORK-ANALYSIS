# SOCIAL-NETWORK-ANALYSIS
# Social Network Analysis Project
## Overview
This project focuses on analyzing complex social network structures using graph theory and network science techniques. The objective is to extract meaningful insights such as influential nodes, community structures, and connectivity patterns from network data.
## Key Objectives
- Understand the structure of social networks
- Identify influential nodes using centrality measures
- Detect communities within the network
- Visualize network relationships effectively
## Concepts Used
- Graph Theory
- Network Analysis
- Centrality Measures:
  - Degree Centrality
  - Betweenness Centrality
  - Closeness Centrality
- Community Detection Algorithms
- Network Visualization
## Tools & Technologies
- Python
- NetworkX
- Matplotlib / Seaborn
- Pandas
- Jupyter Notebook
## Project Workflow
1. Data Collection / Dataset Import
2. Data Preprocessing
3. Graph Construction
4. Network Analysis
5. Community Detection
6. Visualization & Interpretation
## Key Insights
- Identification of key influential nodes in the network
- Discovery of tightly connected communities
- Understanding network density and structure
